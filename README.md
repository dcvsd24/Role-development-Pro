当前脚本的功能。自动识别当前目标角色等级。当前背包的材料。根据当前背包材料计算升级到目标等级，仍然需要的材料，中途使用材料不影响计算。
自动刷取的材料包含突破特产如琉璃百合等，怪物材料如丘丘人面具，boss材料，武器突破素材和两种怪物材料。


设置角色需要突破的等级，升到90级就是80级突破，也就是需要80级突破材料。只是配置里没写突破两个字。
设置角色的天赋书类型，如神里凌华使用的是风雅。设置天赋天赋书秘境名称。设置武器材料秘境名称，设置角色等级突破boss材料的名称。
设置天赋目标等级，按照普通攻击，元素战绩，元素爆发，的顺序排列。示列1-9-9，普通攻击1级，元素战绩9级，元素爆发9级。
设置武器突破目标等级，你想升到多少级就填多少级。80级指的是80级突破，也就是可以升到90级。 


                                           ⚠️ ⚠️ ⚠️ 
第一次运行前请先鼠标右键点击脚本。打开脚本目录双击运行SymLink.bat。建立pathing文件链接。
建立文件链接后，后续启动不再需要运行SymLink.bat，仅运行一次就可以关联AutoPathing，更新的地图追踪文件也会自动刷新。
运行前请确保已订阅地图追踪，敌人与魔物和地区特产的所有路径脚本。                                          
设置好脚本配置后，脚本会识别角色等级,武器等级，角色突破材料，武器突破材料,角色天赋材料,中途使用使用不影响。
设置角色名称，角色名称需要手动输入，后续运行会自动识别，老角色可以填入别名，例如茜特菈莉，别名奶奶，新角色必须使用标准名称，也就是角色原来的名字。
设置角色的元素类型，用于快速筛选角色必须填写。
设置采集队伍和战斗队伍名称,采集队伍有些材料可能需要水元素,雷元素,火元素等,还有草神采集材料。 
如果你没有草神，可以勾选，跳过草神路线，避免无效路径。










物品刷取完成后，下次运行会直接跳过并通知，刷取过程中出现问题也会通知，所以强烈建议打开JS消息通知。

目前只刷取角色突破素材、天赋书、技能培养素材，武器材料，不会升级以及合成(区域特产、怪物掉落物暂时不考虑添加)，建议额外在一条龙中添加自动副本刷取圣遗物和每日领奖兜底。

脚本的基本运行逻辑是，解析培养方案后，优先查看体力，体力充足的情况下，检查物品数量，物品不够，将会前往对应的副本刷取(考虑了合成时额外产出的情况，实际物品可能会少一些)，副本没有开启，检测后会跳过，当材料刷满后会发送通知。

使用方法：打开米游社找到养成计算器，选择需要培养的角色，确认好培养方案后，不知道 boss 叫什么名字的，点击计算器里的 boss 掉落物物图标即可前往查看。


配置组设置注意事项 1、开启地图追踪行走配置 2、关闭自动拾取 3、开启战斗配置 4、战斗策略选择"根据队伍自动选择" 5、关闭自动拾取 6、其他默认就行，练度较低的话，战斗超时时间可以加长

其他配置需要注意的有，副本刷取调用的是独立任务中的配置，请在独立任务中的自动秘境选择战斗的队伍，但请不要选择任何副本，要刷圣遗物请选择一条龙中的自动秘境配置。
